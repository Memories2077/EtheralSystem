TheDogAPI Usage Guide
Base URL: https://api.thedogapi.com/v1
Authentication: Hầu hết các endpoint yêu cầu truyền header x-api-key. Một số endpoint GET công khai (như /images/search, /images/{id}) không bắt buộc API Key nhưng vẫn nên truyền để ghi lại analytics.

1. Images Endpoints
Data Model: Image Schema
json
{
  "id": "string",
  "url": "string (URI format)",
  "width": "integer",
  "height": "integer",
  "species_id": "string",
  "breeds": [ { /* Breed Object */ } ],
  "categories": [ { /* Category Object */ } ],
  "unsafe_reasons": ["string"]
}

**A. Search Images**
- Method: `GET /images/search`
- Description: Tìm kiếm ảnh công khai. Không yêu cầu API Key.
- Query Params:
  - `limit` (integer, optional): số kết quả trả về, từ 1 đến 100, default 1
  - `page` (integer, optional): trang hiện tại, zero-based, default 0
  - `order` (string, optional): `ASC`, `DESC`, hoặc `RANDOM`
  - `species_id` (string, optional): lọc theo species
  - `breed_id` (string, optional): lọc theo breed
  - `breed_groups` (string, optional): danh sách breed groups phân cách bởi dấu phẩy
  - `exclude_breed_groups` (string, optional): danh sách breed groups cần loại trừ
  - `has_breeds` (boolean, optional): chỉ trả về ảnh có thông tin breed
  - `include_breeds` (boolean, optional): đính kèm thông tin breed trong response
  - `include_categories` (boolean, optional): đính kèm thông tin category trong response
  - `sub_id` (string, optional): lọc theo sub_id
  - `account_id` (string, optional): lọc ảnh thuộc account cụ thể
  - `lang` (string, optional): mã ngôn ngữ 2 ký tự, ví dụ `en`
- Response 200 OK: Array of Image objects.

**B. Get an Image by ID**
- Method: `GET /images/{id}`
- Description: Không yêu cầu API Key.
- Response 200 OK: Một Image object.
C. Get Image Processing Status
•        Method: GET /images/{id}/status
•        Description: Trả về trạng thái xử lý hiện tại của ảnh. Yêu cầu API Key. Poll endpoint này cho đến khi status chuyển sang labelled hoặc unsafe.
•        Response 200 OK:
json
{
  "id": "string",
  "status": "string (to_be_reviewed | clean | unsafe | flagged | does_not_contain_species | labelled)",
  "updated_at": "string (ISO 8601)"
}
D. Get Image Labels
•        Method: GET /images/{id}/labels
•        Description: Lấy kết quả nhận diện breed mới nhất của ảnh. Yêu cầu API Key.
•        Response 200 OK: Kết quả breed identification mới nhất.
E. Delete an Image
•        Method: DELETE /images/{id}
•        Description: Yêu cầu API Key.
•        Response 200 OK: Image deleted successfully.
F. Request an Image to be Made Public
•        Method: POST /images/{imageId}/public-requests
•        Description: Gửi yêu cầu để ảnh được đưa lên public. Yêu cầu API Key.
•        Request Body:
json
{ "request_message": "string (optional, tối đa 500 ký tự)" }
•        Response 200 OK: ImagePublicRequest object.
G. Get Public Request Status
•        Method: GET /images/{imageId}/public-requests/status
•        Description: Yêu cầu API Key.
•        Response 200 OK:
json
{
  "id": "string",
  "image_id": "string",
  "account_id": "string",
  "status": "string (pending | approved | denied | cancelled)",
  "request_message": "string",
  "admin_message": "string",
  "actioned_at": "string (ISO 8601)",
  "created_at": "string (ISO 8601)",
  "updated_at": "string (ISO 8601)"
}
H. Cancel a Public Request
•        Method: DELETE /images/{imageId}/public-requests
•        Description: Hủy yêu cầu public đang pending. Yêu cầu API Key.
•        Response 204 No Content.

Response trả về BasicBreedDto cho user thông thường hoặc ProBreedDto cho user có feature flag full_breed_schema. Các fields cơ bản:
json
{
  "id": "string",
  "name": "string",
  "species_id": "string",
  "life_span": "string",
  "temperament": "string",
  "origin": "string",
  "country_codes": "string",
  "country_code": "string",
  "description": "string",
  "history": "string",
  "alt_names": "string",
  "wikipedia_url": "string",
  "bred_for": "string",
  "breed_group": "string",
  "reference_image_id": "string",
  "weight": { "imperial": "string", "metric": "string" },
  "image": { "id": "string", "url": "string", "width": "integer", "height": "integer" }

A. List All Breeds
•        Method: GET /breeds
•        Description: Yêu cầu API Key. Default trả về tối đa 1000 breeds.
•        Query Params:
o        limit (integer, optional): từ 1 đến 1000, default 1000
o        page (integer, optional): zero-based, default 0
o        order (string, optional): ASC, DESC, hoặc RANDOM
o        species_id (string, optional): lọc theo species
o        breed_groups (string, optional): danh sách breed groups phân cách bởi dấu phẩy
o        exclude_breed_groups (string, optional): danh sách breed groups cần loại trừ
o        extinct (boolean, optional): lọc theo trạng thái tuyệt chủng, default false
o        lang (string, optional): mã ngôn ngữ 2 ký tự
•        Response 200 OK: Array of Breed objects.
B. Get a Breed by ID
•        Method: GET /breeds/{id}
•        Description: Yêu cầu API Key.
•        Query Params:
o        lang (string, optional): mã ngôn ngữ 2 ký tự
•        Response 200 OK: Một Breed object.
C. Search Breeds by Name
•        Method: GET /breeds/search
•        Description: Yêu cầu API Key.
•        Query Params:
o        q (string, optional): từ khóa tìm kiếm, từ 1 đến 30 ký tự

o        page (integer, optional): zero-based, default 0
o        order (string, optional): ASC, DESC, hoặc RANDOM
o        species_id (string, optional): lọc theo species
o        breed_groups (string, optional): danh sách breed groups phân cách bởi dấu phẩy
o        exclude_breed_groups (string, optional): danh sách breed groups cần loại trừ
o        lang (string, optional): mã ngôn ngữ 2 ký tự
•        Response 200 OK: Array of Breed objects khớp với từ khóa.
D. List Breed Groups
•        Method: GET /breeds/breed-groups
•        Description: Lấy danh sách tất cả breed groups hiện có. Yêu cầu API Key.
•        Query Params:
o        species_id (string, optional): lọc theo species

6. Breed Bans Endpoints
A. List Countries with Breed Legislation
•        Method: GET /breeds/bans/countries
•        Description: Lấy danh sách các quốc gia có luật liên quan đến breed. Yêu cầu API Key.
•        Query Params:
o        species_id (string, optional): lọc theo species
o        animal_type (string, optional): dog hoặc cat
•        Response 200 OK:
json
[{
  "iso_code": "string",
  "name": "string",
  "has_national_legislation": "boolean",
  "legislation_level": "string"
}]
B. Get Breed Bans by Country
•        Method: GET /breeds/bans/country
•        Description: Lấy toàn bộ thông tin ban/restriction của một quốc gia. Nếu không có luật cấp quốc gia, trả về luật cấp jurisdiction (tiểu bang/thành phố). Yêu cầu API Key.
•        Query Params:
o        country (string, bắt buộc): mã quốc gia ISO 3166-1 alpha-2, ví dụ AU
o        limit (integer, optional): từ 1 đến 100, default 10
o        page (integer, optional): zero-based, default 0
o        order (string, optional): ASC, DESC, hoặc RANDOM, default ASC
o        species_id (string, optional): lọc theo species
o        legislation_type (string, optional): breed_specific, characteristic_based, behavior_based, hoặc hybrid
o        approach (string, optional): ban, restriction, permit, hoặc approved_list
o        jurisdiction (string, optional): lọc theo jurisdiction cụ thể trong quốc gia
o        breed_groups (string, optional): danh sách breed groups phân cách bởi dấu phẩy
o        exclude_breed_groups (string, optional): danh sách breed groups cần loại trừ
o        lang (string, optional): mã ngôn ngữ 2 ký tự
•        Response 200 OK: Object chứa country, legislation (array), và jurisdictions (array).
C. Get All Bans for a Specific Breed
•        Method: GET /breeds/bans/breed/{breedId}
•        Description: Lấy danh sách tất cả quốc gia và jurisdiction nơi breed này bị banned, restricted, hoặc cần permit. Yêu cầu API Key.
•        Query Params:
o        species_id (string, optional): lọc theo species
•        Response 200 OK:
json
[{
  "country_iso_code": "string",
  "country_name": "string",
  "jurisdiction_name": "string",
  "ban_type": "string",
  "legislation_name": "string",
  "conditions": "string"
}]
